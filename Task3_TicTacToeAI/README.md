# Task 3 - Tic-Tac-Toe AI

## Description
This project is an AI-powered Tic-Tac-Toe game developed using Python. The AI opponent uses the Minimax algorithm to make optimal moves, providing an unbeatable gameplay experience.

## Features
- Human vs AI gameplay
- AI powered by Minimax algorithm
- Win, lose, and draw detection
- Command-line interface

## Technologies Used
- Python

## How to Run

python tictactoe.py
